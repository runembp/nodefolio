Installing Node

package.json, npm and command line variables


Node Express is a server environment, used in this case, for a single page application serving static HTML files on various routes. It is installed by either of the two following examples, making the use of package.json and Node Packet Manager (npm):

Creating a package.json file with an empty code block 
            package.json:
            {}
And then entering the command in your terminal: 
            npm install express
Another way, if you prefer a little more "hands-on" approach, is by entering the following details into your package.json file: 
            package.json:
            {
                "dependencies": {
                    "express": "4.17.1"
                }
            }
And entering the following terminal command: 
            npm install
If all goes well, Node Express is now installed in a folder in the project root under "node_modules".
